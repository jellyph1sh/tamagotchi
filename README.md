#Tamagotchi

##Project goal

The goal of this project is to create a Tamagotchi application in JFX, you will have to take care of your tamagotchi in order that he grows up. 
He has 5 states :
 1. Egg
 2. Baby
 3. Adult
 4. Old
 5. Dead

##How to start

Run the App in  the com.ynov package
